# [Задача №3 - Больше своих соседей](https://coderun.yandex.ru/selections/quickstart/problems/more-your-neighbors)

## Условие задачи

Дан список чисел. 
Определите, сколько в этом списке элементов, которые больше двух своих соседей. 
Выведите количество таких элементов.

## Формат ввода
Вводится список чисел. Все числа списка находятся на одной строке.

## Формат вывода
Выведите ответ на задачу.

## Ограничения

- Ограничение времени      1 с
- Ограничение памяти       64 МБ
